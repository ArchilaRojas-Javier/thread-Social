# thread-Social
Une application complète simulant un réseau social de type Twitter : les utilisateurs peuvent créer des publications, les commenter et consulter leur fil d’actualité.

Ce projet a été développé à des fins pédagogiques afin d’illustrer les bonnes pratiques : principes SOLID, Docker, CI/CD et architecture propre.

# Stack:

### Backend
- Node.js
- Express
- Sequelize (ORM)
- MySQL / PostgreSQL
- JSON Web Tokens (JWT)
- Jest (testing)

### Frontend
- React 18
- Vite
- TypeScript
- React Router
- Context API (o Zustand)
- Axios

### DevOps
- Docker & Docker Compose
- GitHub Actions
